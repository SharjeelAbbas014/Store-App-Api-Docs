# Get all employees
Shows all the employees available in current store

#### Send GET Request 
<Code>
/api/employees/
</Code>

#### With following JSON data
```json
{
"store_id": "String"
}
```

##### Expected Response
```json
    [
    {
        "_id": "String",
        "emp_name": "String",
        "joining_date": "2021-10-07T00:00:00.000Z",
        "store_id": "String",
        "cnic": "String",
        "totalSales": "String",
        "phone": "String",
        "emp_id": "String"
    }
]
```
