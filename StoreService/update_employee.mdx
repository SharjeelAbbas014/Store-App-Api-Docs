# Update employee 
Updates specific employee by id.

#### Send PATCH Request 
<Code>
/api/employees/
</Code>

#### With following JSON data
```json
{
    "_id": "String",
    "emp_name": "String",
    "joining_date": "2021-10-07T00:00:00.000Z",
    "store_id": "String",
    "cnic": "String",
    "address": "String",
    "salary": "String",
    "totalSales": "String",
    "phone": "String",
    "emp_id": "String",
    "emp_mail": "String",
    "emp_password": "String"
}
```

##### Expected Response
```
"Status": "Employee Updated Successfully"
```
