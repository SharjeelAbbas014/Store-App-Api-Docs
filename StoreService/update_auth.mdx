# Update auth 
Updates specific auth by id.

#### Send PATCH Request 
<Code>
/api/auths/
</Code>

#### With following JSON data
```json
{
   "_id": "String",
   "username":"String"
}
```

##### Expected Response
```
"Status": "Auth Updated Successfully"
```
