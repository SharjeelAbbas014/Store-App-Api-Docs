# Update auth 
Updates specific auth by id.

#### Send PATCH Request 
<Code>
/api/auths/
</Code>

#### With following JSON data
```json
{
   "_id": "String",
   "username":"String"
   "lastPasswordEdit": "2021-10-07T00:00:00.000Z",
   "storeId": "String",
   "password": "String"
}
```

##### Expected Response
```
"Status": "Auth Updated Successfully"
```
