# Update store 
Updates specific store by id.

#### Send PATCH Request 
<Code>
/api/stores/
</Code>

#### With following JSON data
```json
{
      "_id": "String",
      "name": "String",
      "totalNumberOfProducts": "String",
      "memberSince": "String",
      "imageUrl": "String",
      "location": "String",
      "city": "String"
}
```

##### Expected Response
```
"Status": "Store Updated Successfully"
```
