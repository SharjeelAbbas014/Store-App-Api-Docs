# Add employee
The store can add employees to the Store.

#### Send POST Request 
<Code>
/api/employees/
</Code>

#### With following JSON data
```json
{
        "emp_name": "String",
        "joining_date": "2021-10-07T00:00:00.000Z",
        "store_id": "String",
        "cnic": "String",
        "address": "String",
        "salary": "String",
        "totalSales": "String",
        "phone": "String",
        "emp_id": "String"
}
```

##### Expected Response

```
"Status": "Employee Added Successfully"
```
