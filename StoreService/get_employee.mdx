# Get employee 
Gets specific employee by id.

#### Send POST Request 
<Code>
/api/employees/get_employee
</Code>

#### With following JSON data
```json
{
   "emp_id": "String"
}
```

##### Expected Response
```json
    {
        "_id": "String",
        "emp_name": "String",
        "joining_date": "2021-10-07T00:00:00.000Z",
        "store_id": "String",
        "cnic": "String",
        "address": "String",
        "salary": "String",
        "totalSales": "String",
        "phone": "String",
        "emp_id": "String",
        "emp_mail": "String"
    }
```
