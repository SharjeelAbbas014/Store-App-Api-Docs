# Get all suggestions

Shows all the receipts available of store

#### Send GET Request

<Code>/api/suggestions/</Code>

#### With following JSON data

```json
{
  "store_id": "string"
}
```

##### Expected Response

```json
{
    "_id": "String",
    "user_id": "String",
    "suggestion_date": "2021-10-07T00:00:00.000Z",
    "message": "String",
    "type": "String",
    "store_id": "String",
}
```
