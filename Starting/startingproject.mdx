# Running The Project
You should have installed [Docker](https://www.docker.com/products/docker-desktop) for any of these things to work

After installing docker clone this [repo](https://github.com/SharjeelAbbas014/Store-App-Backend)

Open the project folder in terminal and run the following command

<Code>
docker-compose up --build
</Code>
