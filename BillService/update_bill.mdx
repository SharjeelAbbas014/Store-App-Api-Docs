# Update bill

Updates specific bill by id.

#### Send PATCH Request

<Code>/api/bills/</Code>

#### With following JSON data

```json
{
  "_id": "String",
  "bill_date": "2021-10-07T00:00:00.000Z",
  "amount": "String",
  "narration": "String",
  "products": [
    {
      "product_name": "String",
      "description":"String",
      "price": "String",
      "quantity": "String",
      "discount": "String",
      "categories": ["String"],
      "subCategories": ["String"]
    }
  ]
}
```

##### Expected Response

```
"Status": "Bill Updated Successfully"
```
