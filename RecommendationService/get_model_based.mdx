# Get Model Based Recommendation
The cashier can add receipts.

#### Send POST Request
<Code>
/api/receipts/
</Code>

#### With following JSON data
```json
{
        "emp_id": "String",
        "receipt_date": "2021-10-07T00:00:00.000Z",
        "amount": "String",
        "narration": "String",
        "products":[{
            "product_name":"String",
            "price":"String",
            "quantity":"String",
            "discount":"String",
            "categories":[
            "String"
            ],
            "subCategories":[
            "String"
            ]
        }]
}
```

##### Expected Response

```
"Status": "Receipt Added Successfully"
```
