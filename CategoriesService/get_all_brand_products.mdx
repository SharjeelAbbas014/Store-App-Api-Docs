# Get All Brand List
Get all specific brand products in a store

#### Send GET Request 
<Code>
/api/categories/brands_products/
</Code>

#### With following JSON data
```json
{
 "store_id": "string",
 "brand": "string"
}
```

##### Expected Response
```json
[
    {
        "Id": "ssssjjs",
        "Name": "newName",
        "Price": "2",
        "InStock": "2",
        "Discount": "10",
        "Categories": [
            "asdkl",
            "asdsa"
        ],
        "Tags": [
            "asdas"
        ],
        "Brand": "Coke",
        "_id": "6299a614320f47a1ba3320db",
        "Description": "string"
    }
]
```
