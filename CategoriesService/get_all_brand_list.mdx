# Get All Brand List
Get all brand list in a store

#### Send GET Request 
<Code>
/api/categories/brands_list/
</Code>

#### With following JSON data
```json
{
 "store_id": "string"
}
```

##### Expected Response
```json
[
    "brand1",
    "brand2"
]
```
