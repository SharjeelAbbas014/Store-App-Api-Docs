# Get Categories Name
Get names of already stored categories of the logged in store

#### Send POST Request 
<Code>
/api/categories/names
</Code>

#### With following JSON data
```json
{
 "store_id": "string"
}
```

##### Expected Response
```json
    ["cat1", "cat2"]
```
