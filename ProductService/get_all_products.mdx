# Add all product by store
The store can get product from its catalogue. Store must be login to make request to get product from the catalogue.

#### Send GET Request 
<Code>
/api/product/all
</Code>

#### With following JSON data
```json
{
}
```

##### Expected Response
```json
    "name": "String",
    "id": "String",
    "storeid": "String",
    "price": "float",
    "discount": "float",
    "brand": "String",
    "categories": String[],
    "tags": String[],
    "location": "String",
    "inStock": "int",
    "totalSold": "int",
    "lastStockAddition": "date"
```
