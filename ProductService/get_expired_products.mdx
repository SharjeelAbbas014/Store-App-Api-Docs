# Get Expired Products
The store can get list of all products that are expired. Store must be login to make request to get expired products

#### Send GET Request 
<Code>
/api/product/expired
</Code>

#### With following JSON data
```json
{
}
```

##### Expected Response
```json
{
[
    "name": "String",
    "id": "String",
    "storeId": "String",
    "price": "float",
    "discount": "float",
    "brand": "String",
    "categories": String[],
    "tags": String[],
    "location": "String",
    "inStock": "int",
    "ExpiryDate": "2021-10-03 12:13",
    "totalSold": "int",
    "lastStockAddition": "2021-10-03 12:13",
    "description": "string"
]
}
```
