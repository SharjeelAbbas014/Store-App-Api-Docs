# Remove a product by store
The store can remove product from its catalogue. Store must be login to make request to remove product from the catalogue

#### Send POST Request 
<Code>
/api/product/delete
</Code>

#### With following JSON data
```json
{
"id": "string",
}
```

##### Expected Response
```json
    "Status": "Product Removed Successfully"
```
