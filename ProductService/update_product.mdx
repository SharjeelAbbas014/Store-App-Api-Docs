# Update a product by store
The store can update product from its catalogue. Store must be login to make request to update product from the catalogue

#### Send POST Request 
<Code>
/api/product/update
</Code>

#### With following JSON data
```json
{
"name": "String",
"id": "String",
"price": "float",
"discount": "float",
"brand": "String",
"categories": String[],
"tags": String[],
"location": "String",
"inStock": "int",
"ExpiryDate": "2021-10-03 12:13",
    "totalSold": "int",
    "lastStockAddition": "2021-10-03 12:13",
    "description": "string"
}
```

##### Expected Response
```json
    "Status": "Product Updated Successfully"
```
