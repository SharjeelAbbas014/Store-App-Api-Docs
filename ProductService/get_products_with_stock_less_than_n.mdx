#  Get product with less stock than n
The store can get all the products that have stock less or equal to the number specified

#### Send POST Request 
<Code>
/api/product/stock_less
</Code>

#### With following JSON data
```json
{
"Number_of_products": "int",
}
```

##### Expected Response
```json
[
{
    "name": "String",
    "id": "String",
    "storeid": "String",
    "price": "float",
    "discount": "float",
    "brand": "String",
    "categories": String[],
    "tags": String[],
    "location": "String",
    "inStock": "int",
    "ExpiryDate": "2021-10-03 12:13",
    "totalSold": "int",
    "lastStockAddition": "2021-10-03 12:13"
}
]
```
