# Add a product by store
The store can increase the product count by a specific number 

#### Send POST Request 
<Code>
/api/product/decrease_product_count
</Code>

#### With following JSON data
```json
{
"Id": "String",
"Increase_by": "Int"
}
```

##### Expected Response
```json
    "Status": "Product Count Increased"
```
