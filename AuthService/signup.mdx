# Signup User

User, Admin of Store and Employee account can be created

#### Send POST Request

<Code>/api/users/signup</Code>

#### With following JSON data

```json
{
  "roleName": "Don't include for customer account|EMP|ADMIN",
  "email": "string",
  "password": "string",
  "method": "Incase of user place 'EMAIL' else don't include this field"
}
```

##### Expected Response

```json
{
  "message": "User Created"
}
```
